The Instructions

Joining A Game

To join a game, click its name in the lobby. If there are no games in the lobby, then click on the "Add Game" button. This will add a new game with a silly randomly generated name.

If you join a game that does not already have cards on the table, click the "(Re)Start Game" buttion to deal the first 12 cards.

Playing A Game

Here is what a game of set looks like:

To play, you must find "sets" on the table and collect them before anyone else does. A "set" is a collection of 3 cards satisfying certain properties that we will describe later.

To pick up a set, just click the three cards that make the set. If the three cards really do make a set, then you will get a point and the server will automatically deal 3 more cards in the place where you picked up the set.

It is possible that the 12 cards on the table do not form a set. In this case, you can click on "Add Cards" to add 3 more cards at a time. Since a table of 21 cards is guarenteed to contain a set, you may not add cards to a table of 21 cards. When you do find a set in a table with more than 12 cards, the server will not deal more cards. It will instead rearrange the remaining cards into a neat rectangle.

When there are no cards left in the deck, the server stops dealing cards. Try to find as many sets as possible in the remaining cards on the table. Once you cannot find any more sets, click "Add Cards" to end the game.

What Is A Set?

Each card has 4 different properties. Here are some examples.

Card Color Shading Shape Number
Red Solid Squiggly 1
Green Empty Diamond 2
Purple Shaded Oval 3

A "color-set" is a collection of 3 cards such that either all 3 cards have the same color or all 3 cards have a different color. Similarly, a "shading-set" is a collection of 3 cards such that either all 3 cards have the same shading or all 3 cards have a different shading. And "shape-sets" and "number-sets" are defined similarly.

A "set" of cards is a collection of three cards that is a color-set, a shading-set, a shape-set, and a number-set all at the same time. For example, the collection of three cards in the above example is a "set" because all the colors are different, all the shadings are different, all the shapes are different, and all the numbers are different.
